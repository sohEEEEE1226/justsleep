🌙 청소년 건강 & 생활시간 대시보드

청소년의 수면시간·주관적 건강인지율 변화와 서울시민 필수생활시간을 분석한 Streamlit 웹앱입니다.

🔗 배포 링크: (https://justsleep-gvhfm4ztnbdyrjzxeave9a.streamlit.app/%EC%88%98%EB%A9%B4%EA%B1%B4%EA%B0%95%EB%B6%84%EC%84%9D)

👥 팀원: 장소희, 장민지, 이호연


📌 프로젝트 소개

질병관리청 청소년건강행태조사(2007~2025)와 서울특별시 생활시간조사(1999~2024) 데이터를 활용하여, 청소년의 수면 부족 현상이 건강 인식에 어떤 영향을 주는지, 그리고 시민 전체의 필수생활시간(수면·식사·위생관리 등)이 25년간 어떻게 변화했는지를 분석했습니다.

🎯 주요 기능

페이지내용🏠 종합 대시보드핵심 지표(KPI) 카드, 수면·건강인지율 추이, 생활시간 구성 변화, 상관관계 분석🌙 수면 & 건강인지율연도·성별 필터링, 수면시간/건강인지율 증감 추이, 성별 비교, 피어슨 상관계수⏰ 필수생활시간성별·요일 필터링, 항목별 시간 추이, 요일별 비교, 구성 비율 분석

📊 사용 데이터

파일명출처기간설명청소년.xlsx질병관리청 청소년건강행태조사2007~2025 (19개 연도)주중 평균 수면시간·주관적 건강인지율의 2007년 대비 증감 (전체/남학생/여학생)청소년생활시간.xlsx서울특별시 생활시간조사1999~2024 (5년 주기, 6개 시점)수면, 식사 및 간식, 개인건강관리, 개인위생 및 외모관리 등 필수생활시간 (요일평균/평일/토요일/일요일 × 전체/남자/여자)

🛠️ 기술 스택


Streamlit — 웹앱 프레임워크 및 기본 내장 차트(line_chart, area_chart, bar_chart, scatter_chart)
Pandas — 데이터 전처리 및 분석
openpyxl — 엑셀 파일 읽기



별도의 시각화 라이브러리(matplotlib, plotly 등) 없이 Streamlit 기본 그래프만 사용했습니다.



📁 폴더 구조

your-repo/
├── app.py                  # 메인 앱 코드 (단일 파일)
├── requirements.txt        # 의존성 목록
├── 청소년.xlsx              # 청소년 수면·건강인지율 데이터
└── 청소년생활시간.xlsx       # 서울시민 필수생활시간 데이터

🔍 주요 인사이트


청소년 수면시간은 2017년 이후 2007년 기준치보다 지속적으로 감소하는 추세이며, 여학생의 감소폭이 남학생보다 큽니다.
주관적 건강인지율은 2015년 정점 이후 하락 반전했고, 여학생은 2021년부터 기준치 이하로 진입했습니다.
수면시간 변화와 건강인지율 변화 사이에는 양의 상관관계가 관찰됩니다.
서울시민 필수생활시간은 1999년 대비 2024년 약 65분 증가했으며, 수면이 전체의 약 70%를 차지합니다.
일요일 수면시간이 평일보다 길게 나타나, 주말에 부족한 수면을 보충하는 패턴이 확인됩니다.
